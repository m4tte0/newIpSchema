Usage
=====

Installation
------------

To use Lumache, first install it using pip:

.. code-block:: console

   (.venv) $ pip install lumache

----

Naming and Nomenclature
-----------------------

| KB :: knowledgebase
| TN :: technical note
| BL :: bulettin
| WP :: white paper

::

      NNN.MM --> ID
      |||
      || \_ STATUS : 1 = DF | 2 = PP | 3 = AC | 4 = DPR | 5 = RJ | 6 = NV
      | \
      |  \_ TOPIC : 1 = SW | 2 = HW | 3 = NET | SPP
       \
        \_ TYPE : 1 = GEN | 2 = MAC | 3 = NSW
